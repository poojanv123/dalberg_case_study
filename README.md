**Dalberg Case Study Submission**

Submitted by: Pooja Verulkar

Submission package contents:
1.	data: input data provided by Dalberg
2.	output: output files for each task
a.	dist_int_delivery.csv: predicted delivery feasibility for each district-intervention pair based on the ML model in task A
b.	portfolio_allocation.csv: allocated funding for district-intervention pair for task B
c.	top10_codes.json and top10_districts.csv: top 10 districts with highest allocation from task B
d.	plots: visualizations from outputs for task A
3.	task_a_feasibility_model.ipynb, task_b_portfolio_allocation.ipynb and task_c_chatbot.py: Python scripts for all the tasks
4.	Task_A_readme.pdf, Task_B_readme.pdf and Task_C_readme.pdf: documentation for each task briefly explaining the workflow, assumptions and instructions.

To open the chatbot ui for task C, run the task_c_chatbot.py file and go to http://localhost:5050 in a browser.
